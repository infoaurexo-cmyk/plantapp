     - router.get('/:userId', ...)
     - router.get('/:analysisId', ...)

  ❌ macOS binaries incompatible with Linux Docker environment
  ❌ npm ci --only=production used prebuilt binaries

Solution Applied:
  ✅ Dockerfile changes:
     FROM node:22-bookworm
     
     RUN apt-get install -y \
         build-essential \    ← C/C++ compiler toolchain
         python3 \            ← Required for node-gyp
         make \               ← Build automation
         g++ \                ← C++ compiler
     
     RUN npm install --only=production  ← Rebuild from source, not prebuilt

  ✅ SQLite3 now compiles fresh for Linux environment

Result: SQLite3 loads successfully in Docker container
